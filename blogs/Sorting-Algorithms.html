Bubble Sort

Bubble sort is a kind of exchange sort. The idea of the bubble sort is to compare each adjacent elements, and order them properly until all the items are in order.

Best case: n - 1 comparisions if everything is in order.

Worst case: (n - 1) / 2 comparisions if everything is in reversed order.

Time Complexity: O(n ^ 2)
