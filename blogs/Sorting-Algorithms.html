Bubble Sort

Bubble sort is a kind of exchange sort. The idea of the bubble sort is to compare each adjacent elements, and order them properly until all the items are in order.

Best case: n - 1 comparisions if everything is in order.

Worst case: (n - 1) / 2 comparisions if everything is in reversed order.

Time Complexity: O(n ^ 2)

Selection Sort

Selection sort select the minimal (or maximal) element in the unordered part, then put it at the front of the rest of items. Then, keep searching for minimal (or maximal) element in the remaining items, until everything is in order.

Best case: 0 comparisions if everything is in order.

Worst case: (n - 1) comparisions if everything is in reversed order.

Time Complexity: O(n ^ 2)

Insertion Sort

Insertion sort traverse the whole unordered sequence, find correct position for each element.

Best case: n - 1 comparisions if everything is in order.

Worst case: (1/2)n(n - 1) comparisions if everything is in reversed order.

Time Complexity: O(n ^ 2)

Shell Sort

Shell sort can insert element by a large gap, then reduce the gap gradually. When the gap is one, then it has no difference with insertion sort. The time complexity of shell sort depends on the choice of step.
